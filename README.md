# Apollo Client GraphQL

## Description
A simple client application built with Apollo Client to interact with a GraphQL API.

## What's Built
- A React application using Apollo Client to perform queries and mutations.
- Basic UI to display data and interact with the API.

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/apollo-client-graphql.git
   cd apollo-client-graphql
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm run dev
   ```
4. Open your browser and navigate to `http://localhost:3000`.

## API Documentation
- **GET /graphql**: Endpoint to interact with the GraphQL API.

## Environment Variables
- `NEXT_PUBLIC_GRAPHQL_ENDPOINT`: URL for the GraphQL API (e.g., `http://localhost:4000/graphql`).
